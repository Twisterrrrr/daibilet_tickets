# SeoMeta Spec

Универсальный SEO-блок для всех сущностей (City, Event, Venue, Landing, Article, Combo).

## Entity types

- `CITY` | `EVENT` | `VENUE` | `LANDING` | `ARTICLE` | `COMBO`

## Приоритет источников

1. **Manual** — SeoMeta с `autoGenerated = false` перекрывает всё
2. **Auto** — SeoMeta с `autoGenerated = true`
3. **Default** — вычисляемый fallback из полей сущности

## Поля

- `title` — meta title
- `description` — meta description
- `h1`
- `canonicalUrl`
- `robots` (например `index,follow` / `noindex,nofollow`)
- `ogTitle`, `ogDescription`, `ogImage`
- `jsonLd` — JSON-объект для `<script type="application/ld+json">`

## Правила

- Если `canonicalUrl` пусто → не выводить canonical tag
- Если `robots` пусто → default `index,follow`
- Если `jsonLd` пусто → не выводить JSON-LD script
