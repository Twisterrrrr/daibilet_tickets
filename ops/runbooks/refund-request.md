# Runbook: Запрос на возврат

## Симптом

Клиент просит вернуть деньги за заказ.

## Проверки

1. **Админка → Orders:** статус заказа (PAID, FULFILLED)
2. **PaymentIntent:** есть ли `providerPaymentId` для YooKassa
3. **Политика возврата:** по правилам (полный / частичный / нет возврата)

## Диагностика

- Заказ в статусе FAILED / REFUNDED — уже обработан
- YooKassa: проверить возможность refund через API (сроки, сумма)

## Действия в админке

1. **Admin → Reconciliation / Payment:** кнопка Refund (если реализовано)
2. **Или:** вызвать RefundService / endpoint для частичного/полного возврата
3. **Обновить статус:** Package → REFUNDED, уведомить клиента

## Эскалация

- YooKassa возвращает ошибку при refund
- Спорный случай (частичный возврат, комиссия)
- Нужно решение от руководства
